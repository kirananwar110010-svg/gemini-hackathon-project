# gemini-hackathon-project
This is an AI-powered application built using the Gemini 3 API for the Gemini Hackathon. It helps creators and small businesses generate content ideas and create a 30-day social media plan. By automating content planning, the app saves time, improves consistency, and simplifies social media marketing.
